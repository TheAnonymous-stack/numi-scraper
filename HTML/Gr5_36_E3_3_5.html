Is k = 3 a solution to the inequality below?

Choices:

A. yes
B. no

Solution:

Step 1/2: Plug k = 3 into the inequality.

Step 2/2: k ≤ 3 3 ≤ 3 Plug in k=3 It is true that 3 ≤ 3. So, k = 3 is a solution.
